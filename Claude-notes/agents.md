# Agents

## אגנטים ששימשו בפרויקט זה

### subagent-driven-development
ביצוע תוכנית task-by-task עם review אחרי כל שלב.
- שולח implementer subagent לכל task
- spec reviewer בודק שהקוד תואם לדרישות
- code quality reviewer בודק איכות קוד
- **מתי להשתמש:** כשיש תוכנית מוכנה ורוצים לבצע בסשן הנוכחי

### writing-plans
כתיבת תוכנית יישום מפורטת לפני כתיבת קוד.
- **מתי להשתמש:** כשיש PRD או דרישות ורוצים לתכנן לפני ביצוע

### systematic-debugging
דיבוג שיטתי של באגים.
- **מתי להשתמש:** כשיש שגיאה שלא מבינים

### requesting-code-review
סקירת קוד מקיפה לאחר סיום פיצ'ר.
- **מתי להשתמש:** לפני העלאה ל-GitHub / לאחר סיום מימוש

### brainstorming
סיעור מוחות לפני בניית פיצ'ר חדש.
- **מתי להשתמש:** כשמתחילים פיצ'ר חדש ורוצים לחשוב על אפשרויות
